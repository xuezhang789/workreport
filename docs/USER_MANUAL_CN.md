# WorkReport 使用教程

WorkReport 是面向研发团队的日报、项目、任务、绩效和团队管理系统。本文档按实际页面与权限整理，适合普通成员、项目经理、管理员和超级管理员使用。

---

## 1. 角色与权限

系统内有两类角色概念：

- 职位角色：开发、测试、产品、设计、运维、项目管理，用于日报字段、团队筛选和统计。
- RBAC 权限角色：项目负责人、项目经理、项目成员、全局管理员，用于控制项目、任务、团队和后台页面权限。

常见权限范围：

- 普通成员：查看参与项目、处理自己的任务、提交日报、管理个人资料和通知。
- 项目负责人/项目经理：管理自己负责项目的阶段、成员、任务、仓库和附件，查看团队日报和统计。
- 管理员：管理项目、任务、日报、团队、模板、绩效、审计等运营功能。
- 超级管理员：拥有系统最高权限，可管理人事档案、合同、薪资、用户角色和高敏感配置。

如果访问页面显示“无权限”，请联系项目负责人或超级管理员确认项目成员关系和 RBAC 角色。

## 2. 注册、登录与账户安全

### 注册

系统采用邀请码注册。

1. 打开注册地址或登录页中的注册入口。
2. 输入管理员提供的邀请码。
3. 填写用户名、邮箱和密码。
4. 如果邀请码限定邮箱，注册邮箱必须与邀请码绑定邮箱一致。

邀请码可能失效、已使用或已过期。遇到提示时，请让邀请人重新生成。

### 登录与退出

- 使用用户名或邮箱登录。
- 登录后默认进入工作台。
- 右上角用户菜单可进入个人中心或退出登录。

### MFA

生产环境建议超级管理员启用 MFA。

1. 首次进入系统后，按页面提示绑定身份验证器。
2. 扫描二维码并输入一次性验证码。
3. 保存恢复码。恢复码只显示一次，应存入组织密码管理器。
4. 手机或验证器不可用时，可使用未用过的恢复码登录。

## 3. 个人中心

入口：右上角头像 -> 个人中心。

可维护内容：

- 基本信息：姓名、头像、职位展示。
- 安全设置：修改密码。
- 邮箱验证：绑定和验证邮箱，用于通知和找回密码。
- 可见性：控制团队成员是否可看到邮箱。
- 通知偏好：重要邮件通知、邮件周报、站内通知。

邮箱验证流程：

1. 输入邮箱。
2. 点击获取验证码。
3. 查收邮件并输入 6 位验证码。
4. 页面显示已验证后生效。

如果提示“未验证 / Unverified”，说明当前邮箱尚未完成验证码确认；如果收不到邮件，请检查垃圾箱，并联系管理员确认 SMTP 配置。

## 4. 工作台

入口：顶部导航 -> 工作台。

工作台聚合每日重点：

- 今日日报状态。
- 我的任务概览。
- 参与项目进度。
- 近期日报。
- 快捷操作：写日报、我的任务、创建任务、项目列表等。

建议每日先查看工作台，再处理任务和日报。管理员可以从工作台快速进入管理视图。

---

## 5. 日报管理

### 写日报

入口：顶部导航 -> 写日报。

填写要点：

- 日期默认当天，可根据权限和业务要求补交。
- 角色决定表单字段，例如开发填写今日工作、问题、明日计划；测试填写测试范围、缺陷和计划。
- 可关联一个或多个项目。
- 支持保存草稿和提交。
- 内容会按结构化字段存储，便于搜索、统计和导出。

模板能力：

- 系统可按角色提供默认提示和示例。
- 模板中心可维护日报模板和任务模板。
- 管理员可在角色模板页面调整不同职位的默认日报提示。

### 我的日报

入口：顶部导航 -> 日报 -> 我的日报。

可操作：

- 按日期查看历史日报。
- 查看详情。
- 编辑草稿或允许修改的日报。
- 导出自己的日报数据。

### 团队日报

入口：管理菜单 -> 日报管理。

适用角色：管理员、项目负责人、项目经理。

可操作：

- 按成员、项目、日期筛选日报。
- 查看缺报人员。
- 导出团队日报。
- 对缺报人员发起提醒。

## 6. 项目管理

### 项目列表

入口：顶部导航 -> 项目。

普通成员可查看自己参与或有权限访问的项目。管理员和有管理权限的用户可新建项目。

常用功能：

- 按名称、代号、负责人、阶段搜索或筛选。
- 查看项目进度、成员数和阶段。
- 导出项目列表。

### 项目详情

项目详情包含：

- 基础信息：项目名称、代号、负责人、开始/结束日期、描述。
- 阶段进度：当前阶段、整体进度、阶段变更记录。
- 成员：负责人、项目经理、普通成员。
- 关联任务：项目下任务列表、状态、负责人、截止时间。
- 最近日报：关联项目的日报动态。
- 附件：项目文档上传、下载和删除。
- 代码仓库：维护 Git 仓库名称与地址。

权限说明：

- 项目负责人和项目经理可管理阶段、成员、仓库和附件。
- 项目成员可查看项目并上传项目附件。
- 附件下载必须经过系统鉴权，不应直接访问媒体目录。

### 阶段管理

项目阶段包括启动、需求、设计、开发、测试、部署、结项等默认阶段。阶段变化会记录历史，并可通知相关成员。

管理员可通过项目阶段配置页面维护阶段名称、排序和默认进度。

---

## 7. 任务管理

### 我的任务

入口：顶部导航 -> 任务。

普通成员可查看自己负责或协作的任务。

支持：

- 按项目、状态、优先级、分类筛选。
- 查看任务详情。
- 更新允许的状态。
- 添加评论和附件。
- 导出任务。

### 管理任务

入口：管理菜单 -> 任务管理。

适用角色：管理员、项目负责人、项目经理。

可操作：

- 新建任务并指定负责人、协作人、项目、分类、优先级和截止时间。
- 批量操作任务。
- 编辑任务。
- 查看任务历史。
- 导出全部或选中任务。

### 任务分类与状态

任务分类：

- 普通任务：TODO -> IN PROGRESS -> REVIEW -> DONE / CLOSED。
- Bug：NEW -> CONFIRMED -> FIXING -> VERIFYING -> CLOSED，可按规则重开。

系统会校验状态流转，避免多人同时编辑覆盖。遇到“任务已被其他人更新”提示时，请刷新页面后重试。

### SLA

SLA 根据截止时间、项目设置和系统阈值计算。

- 临近截止会进入预警。
- 已逾期会出现在统计和绩效看板中。
- Celery 定时任务会处理 SLA 提醒。

管理员可在 SLA 设置或系统设置中维护相关阈值。

## 8. 团队管理

入口：管理菜单 -> 团队管理。

适用角色：超级管理员、拥有可管理项目的项目负责人/项目经理。

页面包含：

- 成员列表。
- 按职位、项目、关键词筛选。
- 成员参与项目标签。
- 权限弹窗，用于维护成员项目关系。
- 项目团队视图，查看每个项目成员、负责人和角色分布。

成员项目授权流程：

1. 在成员列表中点击“权限”。
2. 查看该成员当前已参与项目。
3. 在“添加项目”中选择可管理项目。
4. 点击添加。
5. 如需移除，在已参与项目列表中点击移除。

权限边界：

- 超级管理员可管理所有项目。
- 项目负责人/项目经理只能把成员加入自己可管理的项目。
- 如果项目已停用或当前用户无管理权限，系统会拒绝添加或移除。

角色更新：

- 只有超级管理员可修改成员职位角色。
- 修改角色会影响日报字段、团队筛选和部分统计维度。

## 9. 人事管理

入口：管理菜单 -> 人事管理。

适用角色：超级管理员。

可维护信息：

- 在职/离职状态。
- 入职时间、试用期。
- 试用薪资、正式薪资、币种。
- USDT 地址与收款二维码。
- 中介公司与中介费用。
- 离职日期、备注。
- 合同文件上传、下载和删除。
- 薪资变更历史。

安全说明：

- 薪资、地址、备注等敏感字段使用字段级加密。
- 合同和收款二维码只允许超级管理员通过鉴权接口访问。
- 修改薪资会记录历史，便于审计。

## 10. 统计与绩效

入口：管理菜单 -> 绩效看板 / 任务统计 / 统计页面。

常见视图：

- 任务总数、完成率、逾期数。
- 项目 SLA 概览。
- 项目燃尽图。
- 日报合规率与缺报名单。
- 角色维度和项目维度统计。
- 紧急 SLA 任务列表。

使用建议：

- 项目经理每天关注逾期任务和缺报人员。
- 管理员每周导出绩效数据，用于例会和复盘。
- 发现统计异常时，优先检查任务状态、截止时间、项目成员关系和日报关联项目。

## 11. 模板中心与角色模板

模板中心入口：管理菜单 -> 模板中心。

支持：

- 新建日报模板。
- 新建任务模板。
- 按项目或角色筛选模板。
- 预览模板。
- 复用模板内容。

角色模板入口：管理菜单 -> 角色模板。

支持：

- 配置不同职位的日报提示语。
- 启用或停用默认模板。
- 维护字段占位示例。

初始化模板由管理员或运维执行：

```bash
python manage.py init_role_templates
```

## 12. 通知中心

入口：

- 顶部导航铃铛。
- 管理菜单或通知列表页面。

通知类型：

- 任务分配、任务更新、任务提及、协作人变更。
- SLA 提醒。
- 项目更新、项目成员变更、项目经理变更、负责人任命。
- 日报提醒。
- 系统通知。

操作：

- 点击通知可跳转到对应对象。
- 可标记单条或全部已读。
- 可删除单条或清空已读通知。

通知投递：

- 站内通知通过 WebSocket 实时推送。
- 邮件通知通过 Outbox 和 Celery 异步投递。
- 投递失败会按配置重试，超过次数后进入终止状态。

## 13. 全局搜索

入口：顶部搜索或 `/reports/search/`。

搜索范围：

- 项目。
- 任务。
- 日报。

权限说明：

- 搜索结果会按真实业务权限过滤。
- 无项目权限的用户不会看到对应项目、任务或日报。
- 新增和修改数据会增量同步搜索索引；首次上线或大批量导入后应由运维执行 `rebuild_search_index`。

## 14. 审计日志

入口：管理菜单 -> 审计日志。

适用角色：管理员、超级管理员。

记录范围：

- 登录、登出。
- 创建、更新、删除。
- 访问敏感文件。
- 导出数据。
- 上传附件。
- 项目阶段、成员、仓库、附件等关键操作。

常用操作：

- 按操作者、动作、结果、对象类型、时间筛选。
- 查看操作摘要和详情。
- 导出审计数据。

审计日志用于问题追踪、权限复核和安全合规，不建议普通管理员手动删除。

## 15. 导出与文件下载

支持导出的模块：

- 我的日报。
- 团队日报。
- 项目列表。
- 项目阶段历史。
- 任务列表。
- 任务统计。
- 绩效数据。
- 审计日志。
- 人事档案。

大文件导出会创建导出任务，页面通过导出任务状态查询进度。导出文件有过期时间，过期文件由运行时维护任务清理。

如果下载失败：

- 确认当前账号仍有对应数据权限。
- 确认导出任务已完成。
- 联系管理员检查 `ExportJob.message`、应用日志和文件存储权限。

## 16. 常见问题

### 无法收到邮箱验证码

先检查垃圾邮件，再确认个人中心填写的邮箱是否正确。若仍失败，请管理员检查 SMTP 配置、Celery Worker 和通知 Outbox。

### 项目或任务列表看不到数据

通常是未加入项目或无对应 RBAC 权限。请项目负责人在团队管理中为你添加项目。

### 添加成员项目失败

可能原因：

- 当前操作者不是该项目负责人/经理。
- 项目已停用。
- 成员或项目不存在。
- 选择框未选择项目。

### 附件下载 403 或 404

403 表示无权限；404 表示文件不存在、存储不可用或记录指向的文件已被删除。请联系项目经理或管理员处理。

### 状态流转失败

任务状态有固定流转规则。Bug 与普通任务的状态流不同，请按页面允许的目标状态操作。

### 统计结果和预期不一致

优先检查：

- 任务是否关联正确项目。
- 任务状态是否已更新到完成或关闭。
- 截止时间是否正确。
- 日报是否关联项目。
- 成员是否加入项目。

## 17. 推荐日常流程

普通成员：

1. 登录后查看工作台。
2. 处理我的任务。
3. 在任务中更新状态、评论和附件。
4. 当天下班前提交日报。
5. 查看通知并处理提醒。

项目经理：

1. 查看工作台、项目详情和绩效看板。
2. 跟进逾期任务和缺报人员。
3. 管理项目成员、阶段、任务和附件。
4. 每周导出项目统计和日报数据。
5. 定期复核审计日志中的关键操作。

超级管理员：

1. 维护邀请码、成员职位和项目权限。
2. 维护人事档案、合同和敏感信息。
3. 检查系统通知、邮件投递和审计日志。
4. 配合运维确认备份、MFA、指标和安全配置。
