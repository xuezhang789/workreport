# WorkReport 系统使用说明文档

WorkReport 是一个集成了日报管理、项目协作、任务追踪和绩效统计的企业级研发管理平台。本文档将帮助您快速上手并熟练使用各项功能。

## 1. 快速开始

### 注册与登录
- **邀请码注册**: 系统采用邀请制注册。请联系管理员获取邀请码链接，或直接输入邀请码进行注册。
- **登录**: 使用注册时的用户名或邮箱进行登录。

### 个人中心设置
登录后，点击右上角头像进入 **个人中心 (Personal Center)**：
- **基本信息**: 修改姓名（昵称）、上传个性化头像。
- **安全设置**: 修改登录密码。
- **邮箱验证**: 绑定并验证邮箱，这是接收重要通知和找回密码的唯一途径。
  - 点击“获取验证码”，输入收到的 6 位验证码完成绑定。
  - 支持“更换”操作以修正输错的邮箱。
- **通知偏好**:
  - **即时邮件**: 开启/关闭任务分配、被提及时的实时邮件通知。
  - **邮件周报**: 开启/关闭每周一自动发送的绩效简报。
  - **站内通知**: 开启/关闭页面右上角的实时弹窗提醒。

## 2. 工作台 (Workbench)

工作台是您每天工作的起点，聚合了最核心的信息：
- **今日概览**: 显示今日任务完成度、日报连签天数、逾期任务数。
- **待办任务 (Top 5)**: 按优先级和截止时间排序的紧急任务，支持点击直接跳转。
- **项目燃尽图**: 您参与的项目的实时进度概览。
- **智能引导**: 系统会根据您的角色（开发、测试、经理）和当前状态（如未写日报、有逾期任务）提供操作建议。

## 3. 日报管理

### 撰写日报
- **入口**: 顶部导航栏 -> “日报” -> “写日报”，或工作台的“写日报”快捷入口。
- **自动填充**: 系统会自动拉取您昨日日报中的“明日计划”填充到今日的“今日工作”栏，减少重复录入。
- **关联任务**: 在撰写时可以勾选今日完成的任务，系统会自动生成工作摘要。
- **格式**: 支持 Markdown 语法，方便排版代码块和列表。

### 查看与管理
- **我的日报**: 查看自己的历史日报记录，支持按日期筛选。
- **团队日报**: (权限允许下) 查看所在项目组成员的日报，便于信息同步。
- **补交机制**: 如果某天忘记提交，系统会记录为“缺卡”，支持事后补交（会标记为补交状态）。

## 4. 任务管理

### 任务创建与分配
- **创建**: 在“任务”页面点击“新建任务”。
- **字段**:
  - **指派给**: 选择任务负责人。
  - **优先级**: 高 (High)、中 (Medium)、低 (Low)。
  - **截止时间**: 设定 Due Date，系统将以此计算 SLA。
  - **关联项目**: 任务必须归属于某个项目。

### 看板与列表视图
- **列表视图**: 传统的表格形式，适合批量查看和搜索，支持按状态、负责人、项目过滤。
- **看板视图 (Kanban)**: 类似 Trello 的卡片拖拽界面。
  - 可将任务卡片在 TODO -> IN PROGRESS -> REVIEW -> DONE 之间拖拽流转。
  - 自动更新任务状态。

### SLA 服务等级协议
- 系统内置 SLA 监控机制。
- **倒计时**: 任务卡片上会显示剩余时间（例如 "2h left"）。
- **预警**:
  - **黄色**: 临近截止时间（默认 6 小时内）。
  - **红色**: 已逾期 (Overdue)。
- **SLA 达成率**: 您的按时完成情况将计入个人绩效统计。

## 5. 项目管理

- **项目列表**: 查看所有参与的项目。
- **项目详情**:
  - **概览**: 项目描述、当前阶段、起止时间。
  - **成员管理**: 项目经理可添加/移除成员。
  - **代码仓库**: 管理项目关联的 Git 仓库地址。
  - **附件管理**: 上传和下载项目相关文档。
- **阶段管理**: 项目经理可调整项目所处的阶段（如：开发中、测试中、已上线）。

---

## 6. 统计与绩效

*注：部分功能仅管理员或项目经理可见*

- **绩效看板 (Performance Board)**:
  - 查看团队整体的任务完成率、逾期率。
  - **SLA 达标率**: 核心考核指标。
  - **日报合规率**: 团队成员日报提交的及时性统计。
- **一键催报**: 管理员可查看当日未提交日报的名单，并点击“一键催报”发送邮件提醒。
- **导出报表**: 支持将工时、日报数据导出为 CSV 文件。

---

## 7. 通知系统

- **站内信**:
  - 页面右上角铃铛图标。
  - 收到新任务、日报提醒或被 @ 时，如有未读消息会显示红点。
  - 支持 WebSocket 实时推送，无需刷新页面即可看到新通知。
- **邮件通知**:
  - **即时通知**: 任务指派、状态变更时发送。
  - **周报推送**: 每周一凌晨自动发送上周工作总结到您的邮箱。
  - **验证与安全**: 敏感操作（如修改密码、绑定邮箱）均有邮件验证码保护。
