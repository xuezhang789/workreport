# P3 改进完成记录

本文记录本轮 P3 低风险体验与可维护性改进。已按要求排除“多语言文案集中管理，移除模板中的中英文硬编码”。

## 已完成

### 1. 用户界面偏好

- 在个人中心新增“界面偏好”页签。
- 支持默认每页条数、舒适/紧凑显示密度、减少动效。
- 保存后立即在当前页面生效，并写入 `UserPreference.data.ui`。

### 2. 偏好数据治理

- 为 `UserPreference` 增加结构化读写方法。
- 新增 `core.services.preferences`，统一规范分页大小、界面密度和动效设置。
- 偏好保存 API 增加分区白名单、JSON 类型校验和大小限制，避免任意偏好键膨胀。

### 3. 偏好驱动分页

- 项目列表、我的任务、后台任务列表、我的日报、后台日报、审计日志、人事列表、团队页成员/项目分页默认值统一读取用户偏好。
- URL 显式传入分页参数时仍优先生效，保证分享链接、筛选链接和既有测试语义不变。

### 4. 可访问性与长时间使用体验

- 基础模板输出 `data-density` 与 `data-reduce-motion`。
- 紧凑模式降低按钮与表单控件高度。
- 减少动效模式降低过渡、动画和滚动动画影响，适配长期使用和动效敏感用户。

## 未纳入本轮

- 多语言文案集中管理与模板硬编码文案替换：按本轮要求排除。
- 更重的个性化能力，例如列显示配置、看板布局保存、复杂筛选方案保存：建议作为后续 P3/P4 小迭代继续推进。
